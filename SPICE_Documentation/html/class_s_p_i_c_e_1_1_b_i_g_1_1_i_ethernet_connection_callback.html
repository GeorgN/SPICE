Handles an incoming connection. Handling must be synchronous.

Parameters
uriPathThe path-part of the URI of the request.
uriParametersThe parameters of the URI of the request.
serverAddressThe serverAddress on which the request was received.
soapActionThe SoapAction if there is one. Otherwise string will be empty.
contentA string with the content of the connection.
responseSOAPActionGives the handler a possibility to set a SOAPAction for the response.
responseContentThe responseContent which should be set by the handler.
ethernetEventConnectionFactoryThe eventConnectionFactory of the connection receiving ethernet server. Will be given to created asynchronous commands.
Returns
Returns true if the request could be handled, false if not -> Throw 400 Bad Request
